JavaScript Haqida

📌 JavaScript nima?

JavaScript — bu web saytlarni jonlantiradigan (interaktiv qiladigan) dasturlash tili.

👉 HTML sahifani yasaydi
👉 CSS uni chiroyli qiladi
👉 JavaScript esa unga harakat va aqllilik beradi

🧠 JavaScript nima qiladi?

JavaScript yordamida:

Tugma bosilganda amal bajariladi
Formadagi ma’lumot tekshiriladi
Animatsiyalar ishlaydi
O‘yinlar yaratiladi
Internetdan ma’lumot olinadi
⚙️ Oddiy misol

alert("Salom!");

👉 Ekranda xabar chiqadi

🔥 JavaScript qayerda ishlatiladi?
🌐 Web saytlar
🖥️ Server (Node.js)
📱 Mobil ilovalar
🎮 O‘yinlar
🧠 Juda oddiy tushuncha

👉 JavaScript — bu saytni “tirik” qiladigan til
👉 U foydalanuvchi bilan ishlaydi

🚀 Xulosa

JavaScript — web dasturlashning eng muhim tillaridan biri bo‘lib, saytlarni oddiy matndan jonli va interaktiv tizimga aylantiradi.

Modul Oyna

JavaScriptda modal oynalar (modal windows) — bu foydalanuvchining e’tiborini jalb qilish uchun chiqadigan maxsus oynalar. Ular ochilganda, odatda, asosiy sahifa bilan ishlash vaqtincha to‘xtaydi.

📌 Modal oynalar nima uchun kerak?
Xabar ko‘rsatish
Ogohlantirish berish
Tasdiqlash (confirm)
Foydalanuvchidan ma’lumot olish
🧠 JavaScriptdagi asosiy modal oynalar
1. alert()

Oddiy xabar chiqaradi.

Misol:
alert("Salom!");

👉 Faqat “OK” tugmasi bo‘ladi.

2. confirm()

Foydalanuvchidan tasdiq so‘raydi.

Misol:
confirm("Rozimisiz?");

👉 “OK” va “Cancel” tugmalari bor
👉 Natija: true yoki false

3. prompt()

Foydalanuvchidan ma’lumot kiritishni so‘raydi.

Misol:
prompt("Ismingizni kiriting:");

👉 Input (yozish joyi) chiqadi
👉 Natija: kiritilgan matn

⚙️ Qanday ishlaydi?

Bu oynalar brauzer tomonidan avtomatik chiqariladi va dizayni o‘zgarmaydi. Ular oddiy va tez ishlatish uchun
qulay, lekin zamonaviy saytlarda ko‘pincha o‘rniga custom (o‘zing yasagan) modal oynalar ishlatiladi.

🚀 Xulosa

Modal oynalar — foydalanuvchi bilan tez aloqa qilishning eng oddiy usuli. JavaScriptda alert, confirm va prompt
orqali oson ishlatiladi.

variableslar haqida

📌 Variables (o‘zgaruvchilar) nima?

Variables (o‘zgaruvchilar) — bu JavaScriptda ma’lumotlarni saqlash uchun ishlatiladigan “quti”ga o‘xshaydi.

Ular ichida turli qiymatlar bo‘lishi mumkin:

matn (text)
son (number)
true/false (boolean)
va boshqa ma’lumotlar
⚙️ JavaScriptda variables yozish

JavaScriptda o‘zgaruvchi 3 xil yo‘l bilan yaratiladi:

1. let

O‘zgaradigan qiymat uchun ishlatiladi.

let ism = "Ali";
ism = "Vali";

👉 qiymatni keyin o‘zgartirish mumkin

2. const

O‘zgarmaydigan qiymat uchun ishlatiladi.

const yosh = 15;

👉 keyin o‘zgartirib bo‘lmaydi

3. var (eski usul)

Eski JavaScript usuli, hozir kam ishlatiladi.

var shahar = "Toshkent";

🧠 Misollar

let ism = "Ali";
let yosh = 16;
let boy = true;

👉 Bu yerda:

ism = matn
yosh = son
boy = true/false
📌 Qayerda ishlatiladi?

Variables JavaScriptda deyarli hamma joyda ishlatiladi:

hisob-kitoblar
foydalanuvchi ma’lumotlari
o‘yinlar
web saytlarv
🚀 Xulosa
Variables — bu JavaScriptning eng asosiy qismi. Ular ma’lumotlarni saqlash va boshqarish uchun ishlatiladi.

Data Type

🧠 JavaScript Data Types

JavaScriptda ma’lumotlar 2 xil bo‘ladi:
👉 Primitive (oddiy)
👉 Non-Primitive (murakkab)

🧠 1. Primitive (oddiy turlar)
👉 Bu turlarda faqat bitta narsa saqlanadi

📝 String (matn)
let ism = "Ali";
👉 Bu matn saqlaydi
👉 Masalan: ism, shahar, xabar
🧠 Oddiy: “yozuv”

🔢 Number (son)
let yosh = 15;
👉 Sonlarni saqlaydi
👉 Hisob-kitob uchun ishlatiladi
🧠 Oddiy: “raqam”

✔️ Boolean (true/false)
let o‘qiydi = true;
👉 Faqat 2 holat:


true = ha

false = yo‘q

🧠 Oddiy: “ha yoki yo‘q”

🚫 Undefined
let a;
👉 Hech narsa berilmagan
👉 Bo‘sh turibdi
🧠 Oddiy: “hali qiymat yo‘q”

❌ Null
let b = null;
👉 Ataylab bo‘sh qilingan
🧠 Oddiy: “bu yerda hech narsa yo‘q”

🔐 Symbol
👉 Har biri noyob (takrorlanmaydi)
🧠 Oddiy: “unikal belgi”

🧮 BigInt
👉 Juda katta sonlar uchun
🧠 Oddiy: “katta raqamlar”

📦 2. Non-Primitive (murakkab turlar)
👉 Bu turlarda bir nechta ma’lumot bo‘ladi

👤 Object
let odam = {
ism: "Ali",
yosh: 16
};

👉 Bir narsa haqida bir nechta ma’lumot
Masalan: odam (ism + yosh)
🧠 Oddiy: “bitta narsa, ichida ko‘p ma’lumot”

📋 Array
let mevalar = ["olma", "banan", "uzum"];
👉 Ro‘yxat saqlaydi
👉 Ko‘p narsani bir joyga yig‘adi
🧠 Oddiy: “ro‘yxat”

⚙️ Function
function salom() {
console.log("Salom");
}

👉 Bir amalni bajaradi
👉 Kodni qayta-qayta ishlatish uchun
🧠 Oddiy: “buyruq (ish bajaradi)”

🚀 Eng oson xulosa
🧠 Primitive:
👉 1 ta narsa saqlaydi
👉 (matn, son, true/false)
📦 Non-Primitive:
👉 ko‘p narsa saqlaydi
👉 (obyekt, ro‘yxat, funksiya)

Operatorslar

Operatorlar — JavaScriptda qiymatlar ustida amal bajaruvchi belgilar.

Masalan:

hisoblash
solishtirish
mantiqiy tekshirish
qiymat berish
➕ 1. Arifmetik operatorlar (hisob-kitob)

a + b → qo‘shish (masalan: 10 + 5 = 15)
a - b → ayirish (10 - 5 = 5)
a * b → ko‘paytirish (10 * 5 = 50)
a / b → bo‘lish (10 / 5 = 2)
a % b → qoldiq (10 % 3 = 1)
a ** b → daraja (2 ** 3 = 8)

⚖️ 2. Taqqoslash operatorlari (comparison)

a == b → qiymatlar tengmi (5 == "5" → true)
a === b → qiymat va type tengmi (5 === "5" → false)
a != b → teng emasmi
a !== b → type bilan teng emasmi
a > b → katta
a < b → kichik
a >= b → katta yoki teng
a <= b → kichik yoki teng

🧠 3. Mantiqiy operatorlar (logical)

a && b → AND (ikkalasi ham true bo‘lsa true)
a || b → OR (bittasi true bo‘lsa true)
!a → NOT (teskarisi)

🧾 4. Tayinlash operatorlari (assignment)

a = b → qiymat berish

a += b → a = a + b
a -= b → a = a - b
a *= b → a = a * b
a /= b → a = a / b
a %= b → a = a % b
a **= b → a = a ** b

🔢 5. Increment / Decrement

a++ → 1 ga oshiradi
a-- → 1 ga kamaytiradi

Farqi:
a++ → oldin eski qiymat, keyin oshadi
++a → darrov oshadi

❓ 6. Ternary operator (qisqa shart)

shart ? qiymat1 : qiymat2

Misol:
age >= 18 ? "Katta" : "Kichik"

Agar shart true bo‘lsa → birinchi qiymat
Aks holda → ikkinchi qiymat

Conditional statement

🧠 JavaScript Conditional (Shartlar)

JavaScriptda shart orqali qaror qabul qilinadi:
👉 Agar shart to‘g‘ri bo‘lsa — kod ishlaydi
👉 Aks holda — boshqa kod ishlaydi

🧠 1. if / else
👉 Eng asosiy shart operatori

📝 Misol:
let yosh = 16;

if (yosh >= 18) {
  console.log("Kirish mumkin");
} else {
  console.log("Kirish mumkin emas");
}

👉 if = agar
👉 else = aks holda
🧠 Oddiy: “agar bo‘lsa / bo‘lmasa”

🧠 else if (ko‘p tekshiruv)

📝 Misol:
let ball = 75;

if (ball >= 90) {
  console.log("A");
} else if (ball >= 70) {
  console.log("B");
} else {
  console.log("C");
}

👉 Bir nechta shart tekshiradi
🧠 Oddiy: “ketma-ket tekshiradi”

🧠 2. switch case
👉 Ko‘p variant bo‘lsa ishlatiladi

📝 Misol:
let kun = 2;

switch (kun) {
  case 1:
    console.log("Dushanba");
    break;
  case 2:
    console.log("Seshanba");
    break;
  default:
    console.log("Boshqa kun");
}

👉 case = variant
👉 break = to‘xtatadi
👉 default = hech biri mos kelmasa
🧠 Oddiy: “tanlash”

🧠 3. Ternary operator (? :)
👉 if else ning qisqa yozilishi

📝 Misol:
let yosh2 = 20;

let natija = (yosh2 >= 18) ? "Katta" : "Kichik";
console.log(natija);

👉 ? = agar to‘g‘ri bo‘lsa
👉 : = aks holda
🧠 Oddiy: “qisqa if”

🚀 Eng oson xulosa

🧠 if / else:
👉 Oddiy shartlar uchun

📦 switch case:
👉 Ko‘p variantlar uchun

⚡ ternary (? :) :
👉 Qisqa yozish uchun
