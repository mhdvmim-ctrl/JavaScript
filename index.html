JavaScript Haqida

JavaScript — bu web sahifalarni interaktiv va dinamik qiladigan dasturlash tili. U asosan brauzerda ishlaydi va foydalanuvchi bilan to‘g‘ridan-to‘g‘ri aloqani ta’minlaydi. Masalan, tugma bosilganda amal bajarish, formalarni tekshirish, animatsiyalar yaratish yoki sahifadagi ma’lumotlarni yangilash JavaScript orqali amalga oshiriladi.

JavaScript dastlab faqat frontend uchun yaratilgan bo‘lsa ham, hozirda u server tomonda ham ishlatiladi. Bunga Node.js misol bo‘la oladi. Shu sababli JavaScript bilan to‘liq web ilovalar yaratish mumkin.

Bu til o‘zining soddaligi va keng imkoniyatlari bilan mashhur. JavaScriptda o‘zgaruvchilar, shart operatorlari, sikllar va funksiyalar kabi asosiy dasturlash tushunchalari mavjud. U boshqa tillar bilan ham yaxshi ishlaydi va deyarli barcha zamonaviy web saytlar JavaScriptdan foydalanadi.

JavaScriptni yozish uchun maxsus dasturlar yoki oddiy matn muharrirlari yetarli. Eng ko‘p ishlatiladigan kod muharrirlaridan biri bu Visual Studio Code hisoblanadi.

Xulosa qilib aytganda, JavaScript — web dasturlashning eng muhim va keng tarqalgan tillaridan biri bo‘lib, saytlarni “jonlantirish” uchun ishlatiladi.

Modul Oyna

JavaScriptda modal oynalar (modal windows) — bu foydalanuvchining e’tiborini jalb qilish uchun chiqadigan maxsus oynalar. Ular ochilganda, odatda, asosiy sahifa bilan ishlash vaqtincha to‘xtaydi.

📌 Modal oynalar nima uchun kerak?
Xabar ko‘rsatish
Ogohlantirish berish
Tasdiqlash (confirm)
Foydalanuvchidan ma’lumot olish
🧠 JavaScriptdagi asosiy modal oynalar
1. alert()

Oddiy xabar chiqaradi.

Misol:
alert("Salom!");

👉 Faqat “OK” tugmasi bo‘ladi.

2. confirm()

Foydalanuvchidan tasdiq so‘raydi.

Misol:
confirm("Rozimisiz?");

👉 “OK” va “Cancel” tugmalari bor
👉 Natija: true yoki false

3. prompt()

Foydalanuvchidan ma’lumot kiritishni so‘raydi.

Misol:
prompt("Ismingizni kiriting:");

👉 Input (yozish joyi) chiqadi
👉 Natija: kiritilgan matn

⚙️ Qanday ishlaydi?

Bu oynalar brauzer tomonidan avtomatik chiqariladi va dizayni o‘zgarmaydi. Ular oddiy va tez ishlatish uchun
qulay, lekin zamonaviy saytlarda ko‘pincha o‘rniga custom (o‘zing yasagan) modal oynalar ishlatiladi.

🚀 Xulosa

Modal oynalar — foydalanuvchi bilan tez aloqa qilishning eng oddiy usuli. JavaScriptda alert, confirm va prompt
orqali oson ishlatiladi.

variableslar haqida

📌 Variables (o‘zgaruvchilar) nima?

Variables (o‘zgaruvchilar) — bu JavaScriptda ma’lumotlarni saqlash uchun ishlatiladigan “quti”ga o‘xshaydi.

Ular ichida turli qiymatlar bo‘lishi mumkin:

matn (text)
son (number)
true/false (boolean)
va boshqa ma’lumotlar
⚙️ JavaScriptda variables yozish

JavaScriptda o‘zgaruvchi 3 xil yo‘l bilan yaratiladi:

1. let

O‘zgaradigan qiymat uchun ishlatiladi.

let ism = "Ali";
ism = "Vali";

👉 qiymatni keyin o‘zgartirish mumkin

2. const

O‘zgarmaydigan qiymat uchun ishlatiladi.

const yosh = 15;

👉 keyin o‘zgartirib bo‘lmaydi

3. var (eski usul)

Eski JavaScript usuli, hozir kam ishlatiladi.

var shahar = "Toshkent";

🧠 Misollar

let ism = "Ali";
let yosh = 16;
let boy = true;

👉 Bu yerda:

ism = matn
yosh = son
boy = true/false
📌 Qayerda ishlatiladi?

Variables JavaScriptda deyarli hamma joyda ishlatiladi:

hisob-kitoblar
foydalanuvchi ma’lumotlari
o‘yinlar
web saytlarv
🚀 Xulosa
Variables — bu JavaScriptning eng asosiy qismi. Ular ma’lumotlarni saqlash va boshqarish uchun ishlatiladi.

Operatorslar

📌 Operatorlar nima?

Operatorlar — JavaScriptda qiymatlar ustida turli amallar bajarish uchun ishlatiladigan belgilar yoki buyruqlar.

Masalan: qo‘shish, ayirish, taqqoslash, mantiqiy tekshirish va boshqalar.

➕ 1. Arifmetik operatorlar (hisob-kitob)

let a = 10;
let b = 5;

a + b → 15 (qo‘shish)
a - b → 5 (ayirish)
a * b → 50 (ko‘paytirish)
a / b → 2 (bo‘lish)
a % b → 0 (qoldiq)

⚖️ 2. Taqqoslash operatorlari (comparison)

let a = 10;

a == 10 → true (teng)
a === 10 → true (qat’iy teng)
a != 5 → true (teng emas)
a > 5 → true (katta)
a < 20 → true (kichik)
a >= 10 → true (katta yoki teng)
a <= 10 → true (kichik yoki teng)

🧠 3. Mantiqiy operatorlar (logical)

let a = true;
let b = false;

a && b → false (va / AND)
a || b → true (yoki / OR)
!a → false (inkor / NOT)

🧾 4. Tayinlash operatorlari (assignment)

let a = 10;

a += 5 → 15 (a = a + 5)
a -= 3 → 7
a *= 2 → 20
a /= 2 → 5
a %= 3 → qoldiq

🚀 Xulosa

Operatorlar JavaScriptda:

hisob-kitob qilish
shartlarni tekshirish
mantiqiy qarorlar qabul qilish

uchun ishlatiladi.
