JavaScript Haqida

📌 JavaScript nima?

JavaScript — bu web saytlarni jonlantiradigan (interaktiv qiladigan) dasturlash tili.

👉 HTML sahifani yasaydi
👉 CSS uni chiroyli qiladi
👉 JavaScript esa unga harakat va aqllilik beradi

🧠 JavaScript nima qiladi?

JavaScript yordamida:

Tugma bosilganda amal bajariladi
Formadagi ma’lumot tekshiriladi
Animatsiyalar ishlaydi
O‘yinlar yaratiladi
Internetdan ma’lumot olinadi
⚙️ Oddiy misol

alert("Salom!");

👉 Ekranda xabar chiqadi

🔥 JavaScript qayerda ishlatiladi?
🌐 Web saytlar
🖥️ Server (Node.js)
📱 Mobil ilovalar
🎮 O‘yinlar
🧠 Juda oddiy tushuncha

👉 JavaScript — bu saytni “tirik” qiladigan til
👉 U foydalanuvchi bilan ishlaydi

🚀 Xulosa

JavaScript — web dasturlashning eng muhim tillaridan biri bo‘lib, saytlarni oddiy matndan jonli va interaktiv tizimga aylantiradi.

Modul Oyna

JavaScriptda modal oynalar (modal windows) — bu foydalanuvchining e’tiborini jalb qilish uchun chiqadigan maxsus oynalar. Ular ochilganda, odatda, asosiy sahifa bilan ishlash vaqtincha to‘xtaydi.

📌 Modal oynalar nima uchun kerak?
Xabar ko‘rsatish
Ogohlantirish berish
Tasdiqlash (confirm)
Foydalanuvchidan ma’lumot olish
🧠 JavaScriptdagi asosiy modal oynalar
1. alert()

Oddiy xabar chiqaradi.

Misol:
alert("Salom!");

👉 Faqat “OK” tugmasi bo‘ladi.

2. confirm()

Foydalanuvchidan tasdiq so‘raydi.

Misol:
confirm("Rozimisiz?");

👉 “OK” va “Cancel” tugmalari bor
👉 Natija: true yoki false

3. prompt()

Foydalanuvchidan ma’lumot kiritishni so‘raydi.

Misol:
prompt("Ismingizni kiriting:");

👉 Input (yozish joyi) chiqadi
👉 Natija: kiritilgan matn

⚙️ Qanday ishlaydi?

Bu oynalar brauzer tomonidan avtomatik chiqariladi va dizayni o‘zgarmaydi. Ular oddiy va tez ishlatish uchun
qulay, lekin zamonaviy saytlarda ko‘pincha o‘rniga custom (o‘zing yasagan) modal oynalar ishlatiladi.

🚀 Xulosa

Modal oynalar — foydalanuvchi bilan tez aloqa qilishning eng oddiy usuli. JavaScriptda alert, confirm va prompt
orqali oson ishlatiladi.

variableslar haqida

📌 Variables (o‘zgaruvchilar) nima?

Variables (o‘zgaruvchilar) — bu JavaScriptda ma’lumotlarni saqlash uchun ishlatiladigan “quti”ga o‘xshaydi.

Ular ichida turli qiymatlar bo‘lishi mumkin:

matn (text)
son (number)
true/false (boolean)
va boshqa ma’lumotlar
⚙️ JavaScriptda variables yozish

JavaScriptda o‘zgaruvchi 3 xil yo‘l bilan yaratiladi:

1. let

O‘zgaradigan qiymat uchun ishlatiladi.

let ism = "Ali";
ism = "Vali";

👉 qiymatni keyin o‘zgartirish mumkin

2. const

O‘zgarmaydigan qiymat uchun ishlatiladi.

const yosh = 15;

👉 keyin o‘zgartirib bo‘lmaydi

3. var (eski usul)

Eski JavaScript usuli, hozir kam ishlatiladi.

var shahar = "Toshkent";

🧠 Misollar

let ism = "Ali";
let yosh = 16;
let boy = true;

👉 Bu yerda:

ism = matn
yosh = son
boy = true/false
📌 Qayerda ishlatiladi?

Variables JavaScriptda deyarli hamma joyda ishlatiladi:

hisob-kitoblar
foydalanuvchi ma’lumotlari
o‘yinlar
web saytlarv
🚀 Xulosa
Variables — bu JavaScriptning eng asosiy qismi. Ular ma’lumotlarni saqlash va boshqarish uchun ishlatiladi.

Data Type

🧠 JavaScript Data Types

JavaScriptda ma’lumotlar 2 xil bo‘ladi:
👉 Primitive (oddiy)
👉 Non-Primitive (murakkab)

🧠 1. Primitive (oddiy turlar)
👉 Bu turlarda faqat bitta narsa saqlanadi

📝 String (matn)
let ism = "Ali";
👉 Bu matn saqlaydi
👉 Masalan: ism, shahar, xabar
🧠 Oddiy: “yozuv”

🔢 Number (son)
let yosh = 15;
👉 Sonlarni saqlaydi
👉 Hisob-kitob uchun ishlatiladi
🧠 Oddiy: “raqam”

✔️ Boolean (true/false)
let o‘qiydi = true;
👉 Faqat 2 holat:


true = ha

false = yo‘q

🧠 Oddiy: “ha yoki yo‘q”

🚫 Undefined
let a;
👉 Hech narsa berilmagan
👉 Bo‘sh turibdi
🧠 Oddiy: “hali qiymat yo‘q”

❌ Null
let b = null;
👉 Ataylab bo‘sh qilingan
🧠 Oddiy: “bu yerda hech narsa yo‘q”

🔐 Symbol
👉 Har biri noyob (takrorlanmaydi)
🧠 Oddiy: “unikal belgi”

🧮 BigInt
👉 Juda katta sonlar uchun
🧠 Oddiy: “katta raqamlar”

📦 2. Non-Primitive (murakkab turlar)
👉 Bu turlarda bir nechta ma’lumot bo‘ladi

👤 Object
let odam = {
ism: "Ali",
yosh: 16
};

👉 Bir narsa haqida bir nechta ma’lumot
Masalan: odam (ism + yosh)
🧠 Oddiy: “bitta narsa, ichida ko‘p ma’lumot”

📋 Array
let mevalar = ["olma", "banan", "uzum"];
👉 Ro‘yxat saqlaydi
👉 Ko‘p narsani bir joyga yig‘adi
🧠 Oddiy: “ro‘yxat”

⚙️ Function
function salom() {
console.log("Salom");
}

👉 Bir amalni bajaradi
👉 Kodni qayta-qayta ishlatish uchun
🧠 Oddiy: “buyruq (ish bajaradi)”

🚀 Eng oson xulosa
🧠 Primitive:
👉 1 ta narsa saqlaydi
👉 (matn, son, true/false)
📦 Non-Primitive:
👉 ko‘p narsa saqlaydi
👉 (obyekt, ro‘yxat, funksiya)

Operatorslar

Operatorlar — JavaScriptda qiymatlar ustida amal bajaruvchi belgilar.

Masalan:

hisoblash
solishtirish
mantiqiy tekshirish
qiymat berish
➕ 1. Arifmetik operatorlar (hisob-kitob)

a + b → qo‘shish (masalan: 10 + 5 = 15)
a - b → ayirish (10 - 5 = 5)
a * b → ko‘paytirish (10 * 5 = 50)
a / b → bo‘lish (10 / 5 = 2)
a % b → qoldiq (10 % 3 = 1)
a ** b → daraja (2 ** 3 = 8)

⚖️ 2. Taqqoslash operatorlari (comparison)

a == b → qiymatlar tengmi (5 == "5" → true)
a === b → qiymat va type tengmi (5 === "5" → false)
a != b → teng emasmi
a !== b → type bilan teng emasmi
a > b → katta
a < b → kichik
a >= b → katta yoki teng
a <= b → kichik yoki teng

🧠 3. Mantiqiy operatorlar (logical)

a && b → AND (ikkalasi ham true bo‘lsa true)
a || b → OR (bittasi true bo‘lsa true)
!a → NOT (teskarisi)

🧾 4. Tayinlash operatorlari (assignment)

a = b → qiymat berish

a += b → a = a + b
a -= b → a = a - b
a *= b → a = a * b
a /= b → a = a / b
a %= b → a = a % b
a **= b → a = a ** b

🔢 5. Increment / Decrement

a++ → 1 ga oshiradi
a-- → 1 ga kamaytiradi

Farqi:
a++ → oldin eski qiymat, keyin oshadi
++a → darrov oshadi

❓ 6. Ternary operator (qisqa shart)

shart ? qiymat1 : qiymat2

Misol:
age >= 18 ? "Katta" : "Kichik"

Agar shart true bo‘lsa → birinchi qiymat
Aks holda → ikkinchi qiymat

Conditional statement

🧠 JavaScript Conditional (Shartlar)

JavaScriptda shart orqali qaror qabul qilinadi:
👉 Agar shart to‘g‘ri bo‘lsa — kod ishlaydi
👉 Aks holda — boshqa kod ishlaydi

🧠 1. if / else
👉 Eng asosiy shart operatori

📝 Misol:
let yosh = 16;

if (yosh >= 18) {
  console.log("Kirish mumkin");
} else {
  console.log("Kirish mumkin emas");
}

👉 if = agar
👉 else = aks holda
🧠 Oddiy: “agar bo‘lsa / bo‘lmasa”

🧠 else if (ko‘p tekshiruv)

📝 Misol:
let ball = 75;

if (ball >= 90) {
  console.log("A");
} else if (ball >= 70) {
  console.log("B");
} else {
  console.log("C");
}

👉 Bir nechta shart tekshiradi
🧠 Oddiy: “ketma-ket tekshiradi”

🧠 2. switch case
👉 Ko‘p variant bo‘lsa ishlatiladi

📝 Misol:
let kun = 2;

switch (kun) {
  case 1:
    console.log("Dushanba");
    break;
  case 2:
    console.log("Seshanba");
    break;
  default:
    console.log("Boshqa kun");
}

👉 case = variant
👉 break = to‘xtatadi
👉 default = hech biri mos kelmasa
🧠 Oddiy: “tanlash”

🧠 3. Ternary operator (? :)
👉 if else ning qisqa yozilishi

📝 Misol:
let yosh2 = 20;

let natija = (yosh2 >= 18) ? "Katta" : "Kichik";
console.log(natija);

👉 ? = agar to‘g‘ri bo‘lsa
👉 : = aks holda
🧠 Oddiy: “qisqa if”

🚀 Eng oson xulosa

🧠 if / else:
👉 Oddiy shartlar uchun

📦 switch case:
👉 Ko‘p variantlar uchun

⚡ ternary (? :) :
👉 Qisqa yozish uchun

Type Conversion

🧠 Type Conversion nima?

Juda oddiy qilib:
👉 bu — ma’lumot turini o‘zgartirish

🧠 Misol uchun:
"5" → bu string (matn)
5 → bu number (son)

👉 Ba’zida bizga matn emas, son kerak bo‘ladi
👉 Shunda uni o‘zgartiramiz

🧠 1. JS o‘zi o‘zgartirsa (Implicit)

📝 Misol:
let a = "5" + 3;
console.log(a);

👉 Natija: "53"

❗ Nima bo‘ldi?
👉 JS 3 ni stringga aylantirdi
👉 va qo‘shib yubordi

🧠 Oddiy: JS o‘zi hal qiladi

🧠 2. O‘zing o‘zgartirsang (Explicit)

📝 Misol:
let a = "5";
let b = Number(a);
console.log(b + 3);

👉 Natija: 8

❗ Nima bo‘ldi?
👉 "5" → 5 ga aylandi
👉 keyin hisoblandi

🧠 Eng kerakli funksiyalar

👉 Number() → songa aylantiradi
👉 String() → matnga aylantiradi
👉 Boolean() → true / false qiladi

🧠 Boolean juda oson tushun:

Boolean(1) → true ✅
Boolean(0) → false ❌
Boolean("") → false ❌
Boolean("salom") → true ✅

👉 Bo‘sh narsa = false
👉 To‘ldirilgan narsa = true

🚀 ENG MUHIM XULOSA

👉 "5" + 3 = "53" ❌ (xato tushunish)
👉 Number("5") + 3 = 8 ✅ (to‘g‘ri)

🧠 Esda tut:
👉 JS ba’zida o‘zi o‘zgartiradi
👉 lekin yaxshisi — o‘zing nazorat qil

Loop (Takrorlash turlari)

🧠 Loop nima?

Juda oddiy qilib:
👉 bu — kodni qayta-qayta ishlatish usuli
👉 takroriy ishlarni avtomatlashtiradi

🧠 Masalan:
👉 1 dan 100 gacha son chiqarish
👉 array ichidagi elementlarni chiqarish

🚀 1. for loop (eng ko‘p ishlatiladi)

📝 Misol:
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

👉 Natija: 1 2 3 4 5

🧠 Qanday ishlaydi?
👉 1. boshlanish (let i = 1)
👉 2. shart (i <= 5)
👉 3. o‘sish (i++)

🧠 Qachon ishlatiladi?
👉 aniq necha marta ishlash kerak bo‘lsa

🚀 2. while loop

📝 Misol:
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

👉 Natija: 1 2 3 4 5

🧠 Qanday ishlaydi?
👉 shart TRUE bo‘lsa ishlaydi
👉 FALSE bo‘lsa to‘xtaydi

🧠 Qachon ishlatiladi?
👉 qancha davom etishini bilmasang

🚀 3. do...while loop

📝 Misol:
let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);

👉 Natija: 1 2 3 4 5

🧠 Eng muhim farq:
👉 do while — kamida 1 marta albatta ishlaydi

🧠 Qachon ishlatiladi?
👉 foydalanuvchi kiritishi kerak bo‘lsa

🚀 4. for...in loop

🧠 Bu nima qiladi?
👉 object ichidagi KEY (kalit) larni aylantiradi

📝 Misol:
let user = {name: "Ali", age: 20};

for (let key in user) {
  console.log(key);
}

👉 Natija:
name
age

🧠 Qanday ishlaydi?
👉 object keylarini beradi

🧠 Qachon ishlatiladi?
👉 object bilan ishlaganda

🚀 5. for...of loop

🧠 Bu nima qiladi?
👉 array ichidagi VALUE (qiymat) larni beradi

📝 Misol:
let arr = [10, 20, 30];

for (let value of arr) {
  console.log(value);
}

👉 Natija:
10
20
30

🧠 Qanday ishlaydi?
👉 array ichidagi qiymatlarni chiqaradi

🧠 Qachon ishlatiladi?
👉 array bilan ishlaganda eng qulay

🚀 ENG MUHIM XULOSA

👉 for → aniq son marta
👉 while → shartga bog‘liq
👉 do while → kamida 1 marta
👉 for in → object KEY
👉 for of → array VALUE

🧠 Esda tut:
👉 object = for in
👉 array = for of
👉 takrorlash = loop

Function (Funksiya)

🧠 Function nima?

Juda oddiy qilib:
👉 Function — bu qayta ishlatiladigan kod bloki
👉 Bir marta yoziladi, istagancha ishlatiladi

🧠 Hayotiy misol:
👉 Telefondagi kalkulyator ham functionga o‘xshaydi
👉 Son berasan → natija qaytaradi

🧠 Nega function kerak?
❌ Bir xil kodni qayta-qayta yozmaslik uchun
❌ Kodni tartibli qilish uchun
❌ Kodni oson boshqarish uchun

🚀 1. Oddiy Function

📝 Sintaksis:

function nomi() {
  // kodlar
}

📝 Misol:

function salomBer() {
  console.log("Salom JavaScript");
}

salomBer();

👉 Natija:
Salom JavaScript

🧠 Qanday ishladi?
👉 function salomBer() → function yaratdi
👉 salomBer(); → functionni ishga tushirdi

🧠 Muhim:
👉 Function ishlashi uchun uni chaqirish kerak

🚀 2. Function nega kerak?

❌ Functionsiz:

console.log("Salom");
console.log("Salom");
console.log("Salom");

✅ Function bilan:

function salom() {
  console.log("Salom");
}

salom();
salom();
salom();

🧠 Foydasi:
👉 kod qisqaradi
👉 o‘qish oson bo‘ladi

🚀 3. Parametrli Function

🧠 Parametr nima?
👉 function ichiga yuboriladigan ma’lumot

📝 Misol:

function salomBer(name) {
  console.log("Salom " + name);
}

salomBer("Ali");
salomBer("Vali");
salomBer("Sardor");

👉 Natija:
Salom Ali
Salom Vali
Salom Sardor

🧠 Qanday ishladi?
👉 name → parametr
👉 "Ali" → argument
👉 function ichiga qiymat yuborildi

🧠 Esda tut:
👉 parametr = qabul qiladi
👉 argument = yuboriladi

🚀 4. Bir nechta parametr

📝 Misol:

function qosh(a, b) {
  console.log(a + b);
}

qosh(5, 3);
qosh(10, 20);

👉 Natija:
8
30

🧠 Qanday ishladi?
👉 a = 5
👉 b = 3
👉 5 + 3 = 8

🚀 5. return operatori

🧠 return nima?
👉 functiondan qiymat qaytaradi

📝 Misol:

function kopaytir(a, b) {
  return a * b;
}

let natija = kopaytir(4, 5);

console.log(natija);

👉 Natija:
20

🧠 Juda muhim farq:

❌ console.log() → faqat chiqaradi
✅ return → qiymat qaytaradi

🧠 Misol:

function test() {
  return 100;
}

let son = test();

console.log(son);

👉 Natija:
100

🚀 6. Arrow Function (=>)

🧠 Bu nima?
👉 functionning qisqa yozilishi

📝 Oddiy function:

function salom() {
  console.log("Salom");
}

📝 Arrow function:

const salom = () => {
  console.log("Salom");
}

salom();

👉 Natija:
Salom

🧠 Parametr bilan:

const kvadrat = (son) => {
  return son * son;
}

console.log(kvadrat(6));

👉 Natija:
36

🚀 7. Qisqa Arrow Function

📝 Misol:

const qosh = (a, b) => a + b;

console.log(qosh(10, 5));

👉 Natija:
15

🧠 Nega ishladi?
👉 1 qator bo‘lsa return yozmasa ham bo‘ladi

🚀 8. Function Expression

📝 Misol:

const ayir = function(a, b) {
  return a - b;
}

console.log(ayir(10, 4));

👉 Natija:
6

🧠 Farqi nima?
👉 function o‘zgaruvchiga saqlanmoqda

🚀 9. Default Parameter

📝 Misol:

function salom(name = "Guest") {
  console.log("Salom " + name);
}

salom();
salom("Ali");

👉 Natija:
Salom Guest
Salom Ali

🧠 Qanday ishladi?
👉 qiymat yuborilmasa Guest ishlaydi

🚀 10. Real Hayotiy Misol

📝 Misol:

function yoshHisobla(tugilganYil) {
  return 2026 - tugilganYil;
}

console.log(yoshHisobla(2010));

👉 Natija:
16

🧠 Qanday ishladi?
👉 2026 - 2010 = 16

🚀 11. Function ichida if ishlatish

📝 Misol:

function tekshir(age) {
  if(age >= 18) {
    console.log("Katta");
  } else {
    console.log("Kichik");
  }
}

tekshir(20);
tekshir(15);

👉 Natija:
Katta
Kichik

🚀 ENG MUHIM XULOSA

👉 function → qayta ishlatiladigan kod
👉 parametr → functionga ma’lumot yuboradi
👉 argument → yuborilgan qiymat
👉 return → qiymat qaytaradi
👉 arrow function → qisqa function
👉 function expression → functionni o‘zgaruvchida saqlash

🧠 ESDA TUTISH UCHUN OSON YO‘L

👉 function = ishchi
👉 parametr = buyruq
👉 return = natija

🧠 Misol:
👉 Sen kalkulyatorga 5 va 3 berasan
👉 U hisoblaydi
👉 return orqali javob qaytaradi

🚀 ENG KERAKLI MINI MISOLLAR

📝 Sonni 2 ga ko‘paytirish:

function ikkiBaravar(son) {
  return son * 2;
}

console.log(ikkiBaravar(7));

👉 Natija: 14

📝 Juft yoki toq:

function tekshir(son) {
  if(son % 2 == 0) {
    return "Juft";
  } else {
    return "Toq";
  }
}

console.log(tekshir(8));
console.log(tekshir(5));

👉 Natija:
Juft
Toq
